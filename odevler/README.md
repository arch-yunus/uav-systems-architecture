# odevler/

Bu dizin, her modüldeki pratik görevlerin teslim edildiği yerdir.

## Yapı

```
odevler/
├── modul1/
│   └── pervane_hesabi.md     ← Pervane ve uçuş süresi hesapları
├── modul2/
│   ├── lehim_fotograf.jpg    ← Lehim fotoğrafı
│   └── batarya_olcum.md      ← Hücre gerilim tablosu
├── modul3/
│   └── gps_fix.png           ← GPS fix ekran görüntüsü
├── modul4/
│   └── sitl_ucus.log         ← SITL uçuş logu
└── modul5/
    └── maiden_flight.md      ← İlk uçuş raporu
```

## Teslim Formatı

Her görev için bir `README.md` veya belirtilen formatta dosya ekle.
Soru işaretlerin varsa Issues bölümünü kullan.
